## ET001: SALES ユーザーのログイン成功

**前提条件:** 未ログイン状態。

**手順:**

1. `/login` を開く
2. メールアドレス欄に `sales1@test.com` を入力する
3. パスワード欄に `password123` を入力する
4. 「ログイン」ボタンをクリックする
