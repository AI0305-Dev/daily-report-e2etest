# パスワード変更フロー テスト計画

**Seed:** `e2e/seed.spec.ts`

---

## ET080: SALES ユーザーがパスワードを変更する

**前提条件:** `sales1@test.com` でログイン済み。

**手順:**

1. ヘッダーのユーザー名（「山田太郎」）をクリックしてドロップダウンを表示する
2. 「パスワード変更」をクリックして `/settings/password` を開く
3. 現在のパスワード欄に `password123` を入力する
4. 新しいパスワード欄に `NewPass456` を入力する
5. 新しいパスワード（確認）欄に `NewPass456` を入力する
6. 「変更する」ボタンをクリックする

**期待結果:** `/reports` にリダイレクトされる
