## ET050: MANAGER が提出済み日報の Problem にコメントを送信する

**前提条件:** `manager1@test.com` でログイン済み。山田太郎の SUBMITTED 日報（当日-3）の詳細画面を開いている。

**手順:**

1. Problem セクションのコメント入力欄に「来週までに対応方針を確認してください」と入力する
2. Problem セクションの「送信」ボタンをクリックする

**期待結果:** 入力欄がクリアされる。送信したコメントがコメント一覧に追加表示される
