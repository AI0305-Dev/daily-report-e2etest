# 管理者機能（顧客マスタ・営業マスタ）テスト計画

**Seed:** `e2e/seed.spec.ts`

---

## ET060: 管理者が顧客を追加する

**前提条件:** `manager1@test.com`（isAdmin=true）でログイン済み。

**手順:**

1. ナビゲーションの「顧客マスタ」リンクをクリックして `/admin/customers` を開く
2. 「+ 顧客を追加」ボタンをクリックする
3. 顧客名欄に「新規顧客テスト株式会社」と入力する
4. 住所欄に「東京都千代田区1-1-1」と入力する
5. 備考欄に「テスト用顧客」と入力する
6. 「保存」ボタンをクリックする

**期待結果:** `/admin/customers` にリダイレクトされる。「新規顧客テスト株式会社」が顧客一覧に表示される

**前提条件:** `manager1@test.com`（isAdmin=true）でログイン済み。顧客「テスト顧客A」が存在する。
