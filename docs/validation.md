# バリデーション・エラーメッセージ定義

**バージョン**: 1.0  
**作成日**: 2026-05-18

---

## 1. 共通エラー

システム全体で共通のエラーメッセージ。

| コード | 発生条件             | エラーメッセージ                                                   |
| ------ | -------------------- | ------------------------------------------------------------------ |
| E000   | サーバー内部エラー   | 予期しないエラーが発生しました。しばらくしてから再度お試しください |
| E001   | 未認証でのアクセス   | ログインしてください                                               |
| E002   | 権限なしでのアクセス | この操作を実行する権限がありません                                 |
| E003   | リソースが存在しない | 対象のデータが見つかりません                                       |

## 3. バリデーションのタイミング

| タイミング         | 対象             | 内容                                                  |
| ------------------ | ---------------- | ----------------------------------------------------- |
| 入力中（onBlur）   | フォーム各項目   | 必須・形式チェック                                    |
| 送信時（フロント） | フォーム全体     | 全項目の一括チェック                                  |
| 送信時（API）      | リクエストボディ | Zodスキーマで検証。フロントをバイパスされた場合も含む |

> フロントとAPIの両方でバリデーションを実施する。APIは Zod スキーマで必ず検証し、フロント側のバリデーションを信頼しない。

## 4. エラー表示のルール

| 種別                             | 表示方法                                                                                                                  |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| 項目単位エラー                   | 入力欄の直下に赤字で表示                                                                                                  |
| フォーム全体エラー（E202など）   | フォーム上部にアラートで表示                                                                                              |
| ログイン認証エラー（E104・E105） | ログインフォーム上部にアラートで表示（リダイレクトなし）                                                                  |
| 未認証エラー（E001）             | トースト通知でページ上部に表示し、`/login` にリダイレクト                                                                 |
| 権限エラー（E002）               | トースト通知でページ上部に表示し、ロール別トップページにリダイレクト（SALES → `/reports`、MANAGER → `/manager/reports`）  |
| リソース未存在エラー（E003）     | ページ遷移時: Next.js `notFound()` で 404 ページを表示。API 操作時（更新・削除など）: トースト通知でページ上部に表示      |
| 訪問記録上限（E206）             | 10件に達したら「訪問記録を追加」ボタンを disabled にする（エラーメッセージは表示しない）。APIでも上限超過時に E206 を返す |
| サーバーエラー（E000）           | トースト通知でページ上部に表示                                                                                            |
